Para reconstruir las librerías de ooHG, siga el siguiente procedimiento:

1. Abra una ventana de comandos de Windows en la carpeta C:\OOHG\SOURCE.

2. Ejecute el comando
      "MAKELIB.BAT HB30" para reconstruir utilizando los compiladores Harbour 3.0 y MinGW.
      "MAKELIB.BAT HB32" para reconstruir utilizando los compiladores Harbour 3.2 y MinGW.
      "MAKELIB.BAT XB"   para reconstruir utilizando los compiladores xHarbour    y BCC.

   Tambien es posible hacerlo utilizando MAKELIB30.BAT o MAKELIB32.BAT o MAKELIBXB.BAT en su lugar.
   Para utilizar otros compiladores C deberá ajustar y utilizar uno de los archivos alternativos:
   MAKELIB_BCC.BAT, MAKELIB_PC.BAT o MAKELIB_VC.BAT

   Si la reconstrucción fracasa verifique que el PATH del sistema incluye C:\OOHG y que las siguientes variables de ambiantes no están seteadas:
      HG_HRB
      HG_MINGW
      HG_ROOT
      LIB_GUI
      LIB_HRB

   Si tiene dudas, por favor cree un archivo llamado CLEAN.BAT con el siguiente contenido
      SET PATH=C:\OOHG
      SET HG_HRB=
      SET HG_MINGW=
      SET HG_ROOT=
      SET LIB_GUI=
      SET LIB_HRB=
      CALL MAKELIB.BAT HB32
   y ejecútelo.

Las librerías pueden reconstruirse, además, utilizando la herramientas 'hbmk2' de Harbour.
Vea "Como construir las librerías de ooHG para Harbour 3.x y MinGW usando el utilitario 'hbmk2' de Harbour".
